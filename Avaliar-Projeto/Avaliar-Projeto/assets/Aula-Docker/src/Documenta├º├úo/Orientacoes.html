Abra seu Terminal

No Windows 11, procure por PowerShell ou use o terminal integrado do VS Code. O Antigravity também pode executar esses comandos por você.

Crie a Pasta do Seu Projeto

Tudo começa com uma pasta organizada. Crie o diretório onde você vai trabalhar:

Clique para copiar mkdir "C:\Projetos\NomeDoSeuProjeto"

Navegue até o Diretório

Este é o passo mais importante! Você precisa estar dentro da pasta para que os comandos do Docker funcionem:

Clique para copiar cd "C:\Projetos\NomeDoSeuProjeto"
Dica: Se você estiver usando o VS Code, basta abrir a pasta (File > Open Folder) e abrir o terminal (Ctrl + `).

Provisione o Ambiente

Agora execute o script de provisionamento automático. Ele vai baixar o banco de dados (Bitnami Postgres), o Node 24.13 e configurar o TypeScript.

PowerShell (Windows) .\provision.ps1
Atenção: Certifique-se de que o Docker Desktop esteja aberto e rodando no seu computador.

Verifique se Funcionou

Assim que o terminal mostrar "Server is running", acesse o endereço abaixo no seu navegador:

API URL http://localhost:3333/health

Dúvidas? Os arquivos de configuração (Dockerfile, docker-compose) já foram pré-criados na sua pasta raiz.
